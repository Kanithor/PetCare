Proyecto

PetCare es un artefacto diseñado para satisfacer las necesidades alimenticias de nuestras mascotas, dontándonos de control remoto respecto a horarios de comida del animal junto con varias funciones asociadas para llevar a cabo un 'comedero/dispensario inteligente para mascotas'.
